About Me

We can make it together

I'm an MCA graduate and a passionate MERN stack developer focused on building scalable, efficient, and user-friendly web applications. My journey includes real-world projects like Stitches — an e-commerce clothing web app, an AI-powered Resume Builder with ATS optimization, and a real-time Chat App using Socket.IO and Zustand. These projects reflect my strong grip on both frontend and backend development.

I work mainly with React.js, Node.js, Express.js, MongoDB,Redux, Tailwind CSS, Zustand, Socket.IO, and REST APIs. Known for being hardworking and detail-oriented, I focus on creating clean UIs, seamless data flow, and smooth user experiences. Whether it's starting from scratch or improving an existing app, I bring strong problem-solving skills and dedication to every project I build.
